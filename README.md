# 🚀 智慧 iPOS 餐廳管理系統 (POS + KDS)

這是一個基於 **Serverless** 架構開發的輕量化「餐廳銷售點系統 (POS)」與「廚房顯示系統 (KDS)」。系統採用 React + Firebase 構建，具備極速同步與強大的後端管理功能，特別適合中小型餐飲業快速部署與使用。

## 🌟 系統亮點
- **即時同步 (Real-time Sync)**：利用 Firebase Firestore 實現跨裝置（櫃台、內場、店長手機）秒級同步。
- **Serverless 架構**：無需架設伺服器，直接透過前端連網即可使用。
- **全功能後端**：餐點管理、排序、營業報表 (PDF 匯出)、人員權限管理應有盡有。
- **極簡內場看板**：獨立的 KDS 系統，具備超時顏色警示與自動音效提示。

## 🛠 技術堆疊
- **核心框架**: React 18 (via UMD CDN)
- **資料庫/即時通訊**: Firebase Firestore
- **樣式設計**: Vanilla CSS (現代化動態介面)
- **分析報表**: html2pdf.js

## 📂 系統架構
1.  **前台點餐 (`index.html`)**
    *   PIN 碼安全登入。
    *   直覺的分類點餐介面與客製化標籤。
    *   內建現金計算機與自動找零計算。
2.  **管理後台 (`admin.html`)**
    *   **權限控制**：根據角色 (Admin/Leader/Cashier) 開放功能。
    *   **數據分析**：查看熱銷排行與營收，並匯出營運報表。
    *   **系統設定**：即時調整菜單、KDS 警示時間、收銀方式等。
3.  **廚房看板 (`kds.html`)**
    *   專為內場設計，免登入展示待製訂單。
    *   **動態顏色條**：綠色(準時)、黃色(警告)、紅色(緊急)。

## 🚀 部署與啟動
1.  **Firebase 配置**：
    在 `index.html` 與 `admin.html` 的 `firebaseConfig` 中填入您的專案金鑰。
2.  **直接執行**：
    本系統為純前端架構，可直接透過網頁瀏覽器打開 `index.html` 或將資料夾部署至 GitHub Pages、Vercel。
